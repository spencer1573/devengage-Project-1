# engage-game-project
This project helps developers learn Object Oriented programming.

## What we will use
- TypeScript
- SCSS
- Webpack
- Firebase

## Steps
- fork project
- git clone
- git checkout start
- npm install -g typescript  
- npm install -g webpack
- npm install -g tsd 
- npm install
- npm run-script run
- navigate to http://localhost:8080/webpack-dev-server/index.html

# Copying a website by just looking at a picture of it



## What we will use
- Webpack
- Html
- CSS

## Steps
- fork project
- git clone
- git checkout start
- npm install -g typescript  
- npm install -g webpack
- npm install -g tsd 
- npm install
- npm run-script run
- navigate to http://localhost:8080/webpack-dev-server/index.html
